Welcome

The aim of this blog is to collect a set of tutorials on the basic foundations of data analytics, statistical analysis and machine learning, with the aid of R, Python and Octave.

Tools like Ipython and knitr in R allow to quickly author technical documents with embedded code, figures and results. The possibility to output such technical notes in html format makes it easy to directly edit such contents and host them via GitHub pages.

Since the start of my PhD studies in aerospace engineering, I have cultivated a general interest in high-level programming languages and in data science subjects. I hope to share some of my personal understanding through this blog.

Having said this, happy computing!
