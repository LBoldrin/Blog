Python computing essentials

Strings

Concerning the manipulation of strings in Python, it is opportune to highlight how indexing of string elements starts from 0. Such rule applies to arrays as well. If the reader (n.b. myserlf included) is used to programming languages as MATLAB or is familiar with other softwares for data analysis as R where indexing starts from 1, then it is suggested to keep a careful eye (two even better) on how elements are indexed in Python. In MATLAB and R str[1] refers to the first element of the array or string, whereas in Python it accesses the second element.Having stressed this out, let's assume a generic string to be saved in the variable str. In the trivial example here provided, the hypothetical chosen string reads "Machine learning in Python" and contains 27 characters. Since indexing starts from 0, the last character to be accessed, i.e. "." is picked with the command str[26], or in parametrized form, str[len(str)-1]. In order to extract a subset of the string, the command str[start:end:step] can be adopted.

str="Machine learning in Python."
print("SUMMARY")
print("Input string: "+str)
print("Length of string: %i" %len(str))
print("First and last characters in the string: "+ str[0] +" and "+str[len(str)-1])
print("Drop first word in the string: "+ str.replace("Machine",""))
print("Drop first and last character in the string: "+str[1:len(str)-1])
print("Sample string with a 2-character step: "+str[::2])
